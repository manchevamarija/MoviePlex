# MoviePlex MVC

MoviePlex MVC is a web application built with ASP.NET Core MVC that allows users to search movies via the OMDb API, view details, save movies to a personal list, add notes, and set statuses such as "Want to Watch". All user data is stored in a local database.

## Features

- Search movies by title using the OMDb API
- Display movie details: title, year, genre, description, poster, and IMDb ID
- Save movies to a personal list
- Add personal notes to movies
- Set statuses for movies (e.g., "Want to Watch")
- Data stored securely in a local database

## Technologies

- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Razor Views
- LocalDB
- OMDb API integration
